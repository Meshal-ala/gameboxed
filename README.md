# 🎮 Gameboxed

Your Gaming Diary — Track, Rate, and Review every game you play.

## Deploy to Vercel (Free)

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Sign in with GitHub
4. Click "New Project" → Import this repo
5. Click "Deploy"
6. Done! Your site is live at `gameboxed.vercel.app`

## Run Locally

```bash
npm install
npm run dev
```

## Tech Stack

- React 18
- Vite
- Vanilla CSS-in-JS
